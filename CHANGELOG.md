- fix:
  - 使用并发和流式响应来调用大模型 API 进行翻译，极大提高了翻译速度

  - 整页对比/替换翻译有尝试保留段落内的行级内联样式(比如 p 标签中的 strong)，但效果不佳，尤其是整页翻译

  - 统一了整页翻译、划词翻译、小窗翻译使用同样的请求，可在控制台看到大模型 API 调用请求，方便调试。
